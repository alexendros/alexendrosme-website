# alexendros-me

Landing estatica de marca personal en [alexendros.me](https://alexendros.me).

Next.js 15 App Router + Tailwind CSS v4 + shadcn/ui + static export. Sin backend.

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build   # genera out/
```
